# Advanced Data Analysis Case Studies

This repository contains multiple case studies completed as part of the **Ahmet Çalık Foundation – Advanced Data Analytics Program (ITUSEM)**.

The projects focus on applying **data analysis, visualization, and business insight generation** techniques using real-world datasets from banking and energy sectors.

Each case study follows a structured analytics workflow including:

* Data exploration
* Data cleaning and preparation
* Statistical analysis
* Data visualization
* Business insight generation
* Dashboard development

# Case Study 1 — Credit Card Customer Behavior Analysis (Excel)

This project analyzes **credit card customer data** to understand customer behavior and support data-driven decision making in banking.

The analysis focuses on:

* customer demographics analysis
* credit limit and utilization behavior
* transaction patterns
* churn (attrition) analysis
* customer segmentation

The case study involves cleaning and transforming a dataset containing **10,000+ credit card customers** and generating insights for management decisions. 

Key tasks included:

* Data cleaning and transformation
* Creating calculated metrics
* Customer segmentation
* Pivot table analysis
* KPI tracking
* Scenario analysis
* Interactive Excel dashboard design

Tools used:

* Microsoft Excel
* Pivot Tables
* Advanced formulas
* Dashboard design

# Case Study 2 — Electricity Consumption Data Analysis (Python)

This project analyzes **electricity consumption and payment data** from three districts in Amasya:

* Hamamözü
* Gümüşhacıköy
* Göynücek

The dataset contains electricity consumption and billing information between **2023 and 2025**. 

Key analysis tasks:

### Data Exploration

* inspecting dataset structure
* calculating descriptive statistics
* identifying missing or abnormal values
* calculating unique customer counts per district

### Data Visualization

* consumption distribution by customer class
* monthly electricity consumption trends
* payment distribution by bank branches
* on-time vs late payment analysis
* consumption distribution and outlier detection

### Data Storytelling

* district-level consumption comparison
* customer segmentation based on consumption levels
* payment risk analysis
* business recommendations for collection processes

Tools used:

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

# Case Study 3 — Electricity Consumption Anomaly Detection (Power BI)

This project focuses on detecting **abnormal electricity consumption patterns** in smart meter data.

The objective is to identify operational risks such as:

* potential electricity theft
* meter malfunction
* network irregularities

The dataset includes measurements such as:

* current values (L1, L2, L3)
* voltage values (V1, V2, V3)
* active energy consumption
* reactive power measurements
* time-based load profile data 

Key analysis tasks:

* anomaly detection logic development
* feature engineering for energy indicators
* risk scoring models
* pivot analysis for anomaly distribution
* Power BI dashboard design

Dashboard outputs include:

* anomaly type distribution
* regional risk map
* anomaly trend analysis
* high-risk subscriber identification

Tools used:

* Power BI
* DAX
* Excel
* Data visualization techniques
